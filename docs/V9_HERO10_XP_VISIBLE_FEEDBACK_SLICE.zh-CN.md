## 1. 已实现

| 行为 | 描述 |
|------|------|
| 等级显示 | 选择英雄时显示 `等级 N` |
| XP 显示 | 非最高等级时显示 `XP current/nextThreshold`，最高等级显示 `XP 最高等级` |
| 技能点显示 | 技能点 > 0 时显示 `技能点 N` |
| HUD 刷新 | 等级/XP/技能点变化时选择缓存键更新，HUD 自动刷新 |
