## 1. 运行时变更

### 1.1 HUD 状态面板

- 当 Divine Shield 活跃时，选择面板新增 `神圣护盾生效 Ns` 显示，N 为剩余秒数（向上取整）。
- 每秒自动更新（通过选择缓存键量化 DS 剩余时间）。
- 效果结束后显示消失。

### 1.2 命令卡施放按钮反馈

- 活跃时：按钮禁用，原因显示 `生效中 Xs`（X 为剩余秒数）。
- 冷却时：按钮禁用，原因显示 `冷却中 Xs`（X 为剩余秒数）。
- 魔力不足：按钮禁用，原因显示 `魔力不足`。
- 优先级：活跃 > 冷却 > 魔力不足（活跃时不显示魔力不足）。
- 所有冷却/持续时间每秒自动刷新，无需取消选择再重选。

### 1.3 选择缓存键

- 单选缓存键 (`selKeyFull`) 新增 `dsActiveKey`（量化 DS 剩余秒数），确保 HUD 每秒自动刷新。
- 命令卡缓存键 (`selKey`) 已在 IMPL1B 中包含 `divineShieldKey`，无需额外修改。

### 1.4 未实现

- **无** 粒子效果或视觉特效。
- **无** 状态栏图标。
- **无** 声音。
- **无** AI 使用 Divine Shield。
- **无** CSS 效果系统。
