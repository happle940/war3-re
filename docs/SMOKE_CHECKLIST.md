## 选择模型

- [ ] 左键点击友方单位 → 选中，绿色选中圈
- [ ] 左键点击空白处 → 清除选择
- [ ] 拖拽框选 → 所有框内友方单位被选中
- [ ] Shift + 左键点击未选中友方 → 添加到选择
- [ ] Shift + 左键点击已选中友方 → 从选择中移除
- [ ] 双击友方单位 → 选中屏幕上所有同类型友方单位
- [ ] Tab（混合选择时）→ 切换 primary 子组，命令卡和 HUD 更新
- [ ] Tab 后选中圈同步正确

## 控制组

- [ ] Ctrl + 1（有选中单位）→ 保存到编组
- [ ] 按 1（已保存编组）→ 召回，选择状态正确
- [ ] Shift + 2（已有选择）→ 追加编组到选择，不重复
- [ ] 召回含死亡单位的编组 → 自动清理，不崩溃
- [ ] 召回后 HUD/命令卡正确
- [ ] 召回后编组信息提示显示（底部闪烁）

## 移动队列

- [ ] Shift + 右键地面（单位移动中）→ 追加到队列，不覆盖当前移动
- [ ] 到达第一个目标后 → 自动前往下一个队列目标
- [ ] 普通（非 Shift）右键地面 → 覆盖移动，队列清空
- [ ] 按 S（停止）→ 停止移动，队列清空
- [ ] 选中时黄色菱形标记显示队列目标位置
- [ ] Shift + A → 左键地面 → attackMove 追加到队列（红色菱形）
- [ ] 队列消费后单位正确切换到对应状态（move / attackMove）

## 命令恢复（auto-aggro 中断恢复）

- [ ] 移动中的单位被敌人接近 → 自动反击
- [ ] 反击结束（目标死亡/逃跑）→ 恢复原移动路径
- [ ] 采集中被攻击 → 不触发自动反击（保持采集）
- [ ] 停止命令 → 切断恢复链，不恢复原命令
- [ ] 驻守命令 → 切断恢复链

## 命令兼容

- [ ] 右键金矿（worker + footman 混选）→ worker 采金，footman 移动
- [ ] 右键树（worker + footman 混选）→ worker 伐木，footman 移动
- [ ] 右键敌方 → 全部攻击
- [ ] S（停止）→ 清除所有移动/攻击状态
- [ ] H（驻守）→ 原地驻守，遇敌攻击不追击
- [ ] A → 左键地面 → 攻击移动（红色十字指示器）
- [ ] ESC → 取消所有模式
