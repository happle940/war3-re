## 3. HERO10-IMPL1 起步范围

- **仅** 敌方非建筑单位击杀给 Paladin 分配 XP。
- 英雄击杀 XP 和野怪 XP 为来源边界数据，运行时延后。
- 建筑击杀不给予 XP。
- 死亡英雄不获得 XP。
- 升级增加 `heroLevel`、授予 `heroSkillPoints`，保留 HERO9 复活公式。
